1.  
    设级数 $\sum_{n=1}^{\infty} a_n$ 与 $\sum_{n=1}^{\infty} b_n$ 中，有一个收敛，另一个发散。
    (1) 证明级数 $\sum_{n=1}^{\infty} (a_n + b_n)$ 必发散。
    (2) 若所给的两个级数都发散，那么级数 $\sum_{n=1}^{\infty} (a_n + b_n)$ 是否必发散？请举例说明。

2.  
    用比值判别法或根值判别法判别下列级数的敛散性：
    (1) $\sum_{n=1}^{\infty} \frac{a^n}{\ln(n+1)} \quad (a > 0)$
    (2) $\sum_{n=1}^{\infty} 3^n \cdot \left(\frac{n}{n+1}\right)^{n^2}$

3.  
    利用积分判别法判别下列级数的敛散性：
    (1) $\sum_{n=3}^{\infty} \frac{\ln \ln n}{n \ln n}$
    (2) $\sum_{n=2}^{\infty} \frac{1}{\ln^p n} \quad (p > 0)$ 

4.  
    用适当的方法判别下列级数的敛散性：
    (1) $\sum_{n=1}^{\infty} \sqrt{n} \left( 1 - \cos \frac{\pi}{n} \right)$
    (2) $\sum_{n=1}^{\infty} \left( \frac{1}{n} - \sin \frac{1}{n} \right)^a \quad (a > 0)$

5.  
    利用幂级数的性质求下列级数的和：
    (1) $\sum_{n=1}^{\infty} \frac{n}{2^n}$
    (2) $\sum_{n=1}^{\infty} \frac{1}{(n^2 - 1)2^n}$

6.  
    将下列函数在给定点 $x_0$ 展开成 $(x - x_0)$ 的幂级数，并指出展开式成立的区间：
    (1) $f(x) = \ln(1 + x), \quad x_0 = 1$
    (2) $f(x) = \frac{1}{x^2 - x - 6}, \quad x_0 = 1$


7.  
    把下列周期为 $2\pi$ 的函数展开为 Fourier 级数，并写出级数在 $[-\pi, \pi]$ 上的和函数：
    $$f(x) = \begin{cases} 1, & -\pi < x < 0 \\ -x, & 0 \leqslant x < \pi \end{cases}$$


8.  
    把下列各函数在 $[0, \pi]$ 上展开成正弦级数或余弦级数，并写出级数在该区间上的和函数：
    $$f(x) = \begin{cases} 1, & 0 \leqslant x \leqslant h \\ 0, & h < x \leqslant \pi \end{cases} \quad (\text{展开为余弦级数})$$


9. 求下列微分方程的通解： (1) $y'' + 6y' + 9y = 0$ (2) $y''' - y'' - y' + y = 0$
    

    
10. 判断下列函数组在定义区间内的线性相关性： (1) $e^{-2x}, 3e^{-2x}$ (2) $e^{2x}\cos 5x, e^{2x}\sin 5x$
    




11. 求下列微分方程满足所给初值条件的特解： (1) $y'' - 4y' + 3y = 0, \quad y(1) = 0, \quad y'(1) = 2$ (2) $y'' + 9y = 0, \quad y\left(\frac{\pi}{3}\right) = 0, \quad y'\left(\frac{\pi}{3}\right) = 1$
    

    
12. 已知二阶齐次线性微分方程的一个解，试用 Liouville 公式或常数变易法求方程的另一个线性无关解，并求出方程的通解： (1) $x^2y'' + xy' - y = 0, \quad y_1 = x$ (2) $(2x-1)y'' - (2x+1)y' + 2y = 0, \quad y_1 = e^x$
    


13. 求下列微分方程的通解： (1) $y'' - 3y' = x - 1$ (2) $2y'' + y' - y = 2e^{-x}$
    

    
14. 求下列微分方程满足所给初值条件的特解： (1) $y'' - 3y' + 2y = 5, \quad y(0) = 1, \quad y'(0) = 2$ (2) $y'' + y = 2\cos x, \quad y(0) = 1, \quad y'(0) = 0$
    

    
15. 求下列微分方程的通解： (1) $y'' + 2y' + 2y = e^{-x}\sin x$ (2) $y'' + 2y' + 2y = x + 3 + e^{-x}\sin x$
    

    
16. 设函数 $f(x)$ 连续，且满足 $f(x) = e^x + \int_0^x (t-x)f(t)dt$，求 $f(x)$。
    

17. 求下列函数的一阶差分 $\Delta y_x$：
(1) $y_x = x^2 - 3x + 1$
(2) $y_x = 5^x$

18. 指出下列差分方程的阶数，并判断其是否为线性常系数差分方程：
(1) $y_{x+2} - 4y_{x+1} + 3y_x = 0$
(2) $y_{x+1} = y_x^2 + x$

19. 将含有差分的方程化为不含差分的形式：
已知 $\Delta y_x - 2y_x = 3$，请将其改写为关于 $y_{x+1}$ 和 $y_x$ 的标准差分方程形式。

20. 求下列一阶常系数齐次线性差分方程的通解：
$y_{x+1} - 3y_x = 0$

21. 求满足初始条件 $y_0 = 4$ 的差分方程的特解：
$y_{x+1} = \frac{1}{2}y_x + 6$
